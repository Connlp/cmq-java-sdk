# cmq-java-sdk
Tencent CMQ JAVA SDK

## 获取master代码：
```
git clone https://github.com/tencentyun/cmq-java-sdk.git
```

## 获取指定tag代码，如v1.0.6：
```
git clone --branch v1.0.6 https://github.com/tencentyun/cmq-java-sdk.git
```

## 编译，在工程目录下执行：
```
mvn clean install -Dmaven.test.skip=true 
```

## SDK使用方式：
详情见源码中example
